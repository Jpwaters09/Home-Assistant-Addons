Run your own **Spoolman instance** on Home Assistant to **manage and monitor your spools**.

&nbsp;

## Usage
1. Once the add-on is installed, click on **Open Web UI**.
2. Add your filament spools and configure your inventory.
3. To view filament statistics, inventory, and manage settings, revisit the Web UI at any time.

&nbsp;

## About
### What is this add-on?
This add-on allows you to run Spoolman directly on Home Assistant, enabling you to monitor your filament spools, track usage, and manage inventory efficiently.
